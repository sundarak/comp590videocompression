# Assignment 1

### Overview

This project implements video compression by using 256 arithmetic coding contexts for the coder. I picked the context based on the prior pixel's intensity. I separated the pixels using their brightness vs. darkness and mapped them into 256 independent arithmetic coding contexts.


### Results

Target Compression Rate: 2.5

Achieved Compression Rate: 2.74

*Note: These results are for bourne.mp4 file only.*
